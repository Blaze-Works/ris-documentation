Example Programs

1. Bootloader

Initializes the operating system and displays system information.

VAR BOOT_MSG >> "Booting RIS OS..." PRN $BOOT_MSG MEM SIZE PROC CREATE init SYS DIR INT 0 HLT

2. Interactive Shell

Simulates a command-line interface.

VAR PROMPT >> "RIS> " :loop PRN $PROMPT VAR cmd << PROC CREATE shell SYS DIR GOTO loop HLT

3. Process Monitor

Demonstrates process creation and management.

PROC CREATE main PROC CREATE worker1 PROC LIST PROC KILL 1 PROC LIST HLT

4. File System Explorer

Shows how to manipulate files.

FILE WRITE log.txt "System Log Initialized" FILE READ log.txt FILE DELETE log.txt HLT
